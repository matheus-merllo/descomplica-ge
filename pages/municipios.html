Incluir

O sistema só irá validar os campos "Município/Localidade" e "Cód.IBGE" para gerar notas estaduais e "Cód.Mun.Próprio" para notas municipais.
Cód.Correio: Código do correio.
Município/Localidade: Nome da cidade.
UF: Estado.
Tipo: Município, Distríto ou Povoado.
Codificado CEP: Não é importante para NF.
CEP geral: Não é importante para NF.
População: Não é importante para NF.
DDD: Não é importante para NF.
DDD/discar: Não é importante para NF.
Cód.IBGE: É fundamental que esse código esteja preenchido de forma correta para gerar notas estaduais.
Cód.SIAFI: Não é importante para NF.
Cód.Master maq: Não é importante para NF.
Cód.Mun.Próprio: Poderá ser obrigatorio para notas municipais de prestação de serviço.
