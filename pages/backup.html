Enviar uma cópia para drive(s): Permite salvar uma cópia dos backups gerados em outro disco. Não é possível usar subpastas dentro do disco.

Backups: Selecione o backup que será restaurado.
Restaurar arquivos do módulo: Possibilita restaurar módulos específicos do backup.
Restaurar para append: Ao marcar essa opção, o sistema irá recuperar somente arquivos corrompidos. EX: Se existir algum arquivo corrompido ou em falta, o backup irá usar somente os respectivos dados para preencher a lacuna do sistema.

Backups: Selecione o arquivo de backup e inicie a restauração total.
