keyboard_double_arrow_left Rótulos para embalagens

Os rótulos são maneiras de identificar a embalagem enviada para o cliente.



Título

grade
Nota fiscal: Só é preenchida de forma manual.
Nro pedido: número do pedido interno do cliente. Puxa do campo "Nro pedido cliente" da O.S.
Nro jogos: Só é usado em trabalhos numerados. É o mesmo número de jogos do bloco, por exemplo.
Nro inicial: Só será usado para trabalhos númerados, como blocos. Será informado o número inicial de um intervalo.
Nro final: Só será usado para trabalhos númerados, como blocos. Será informado o número final de um intervalo.
Pacote com: Defini a quantidade em cada pacote.
Total pacotes: Mostrará o total de pacotes dividindo a quantidade pelo "pacote com".
Rótulos extras: Define a quantidade de rótulos extras a sem impressos.
Já impressos: Começa a imprimir a partir da quantidade informada.
Produto: Será vinculado um produto do estoque, caso seja necessário.
Nro prod. cliente: Vincula o número do produto do cliente.
Transportadora: Define a transportadora que fará a entrega.
Peso: Peso total
Data: Data que o rótulo fori emitido. Funciona para controle de validade.
Endereço de entrega: É o endereço que a entrega será feita.
Observações: Acrescenta uma observação na etiqueta.
Saltar primeiros rótulos na impressora: Pulo o número informado de etiquetas, caso o usuario queira. Normalmente é usado quando alguma folha não será usada completamente.
Resto no primeiro pacote: Troca o primeiro pacote pelo resto do produto, caso seja marcado. Se não for marcado, essa quantidade ficará no último rótulo.
Layout: Define um layout padrão que será usado nas etiquetas.
Layout 1/2: Te leva para a página que permite a edição das etiquetas.

Layout

Nesta página é possível definir o layout dos rótulos de acordo com a preferência do usuário.
