As notas devem ser importadas junto com a entrada no estoque.
Para fazer a importação, o nome do arquivo deve ser a chave de acesso do NF do fornecedor. O xml deve ser lançado na pasta raiz "NFe"

grade O CNPJ do recebedor deve ser o mesmo que o cadastrado no sistema para o processo funcionar.

Para importar manualmente, clique em incluir e arraste o xml para dentro da página, verifique o prazo de pagamento e grave o registro. É necessário arrastar o arquivo para dentro da pasta de xml de fornecedores também (C:\NFe\"código da empresa"\"ano"\Forn).

grade O sistema só importa o modelo 55 (DANFE) e 57 (conhecimento de transporte.
